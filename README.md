# VORTEX – AI-Powered Medicine Verification System

## 🚨 Problem Statement

Counterfeit and substandard medicines pose a serious threat to public health. Existing verification systems rely on basic QR codes that can be easily duplicated, making them unreliable and insecure.

---

## 💡 Proposed Solution

ZARIAA is an AI-powered medicine verification system that ensures authenticity through a combination of **secure QR codes, encryption, and intelligent validation**.

The system allows users to scan medicine codes and instantly verify their legitimacy while detecting potential counterfeit products in real time.

---

## ⚙️ Key Features

* Secure QR-based medicine authentication
* AI-based prescription and medicine validation
* Real-time verification from database
* Counterfeit detection alerts
* User-friendly interface for quick scanning

---

## Technology Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, JavaScript
* **AI/ML:** TensorFlow / YOLO (for detection & validation) for future upgrade
* **Hardware (Optional):** ESP32 / Raspberry Pi for future upgrade
* **Security:** Encrypted unique identifiers (QR-based system)

---

## 🏗️ System Architecture

1. Medicine is registered with a unique encrypted QR code
2. User scans the QR code via interface
3. System verifies data from secure database
4. AI module checks prescription and drug validity
5. Result is displayed (Authentic / Suspicious / Counterfeit)

---

## 🔐 Security Approach

Sensitive data such as encryption keys are **not stored in the public repository** and are managed securely using local configuration files.

---

## Impact

* Reduces circulation of counterfeit medicines
